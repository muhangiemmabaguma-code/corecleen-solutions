# CoreCleen Solutions Ltd

Professional cleaning services website built with semantic HTML, responsive CSS, and vanilla JavaScript.

**Live Demo:** https://muhangiemmabaguma-code.github.io/corecleen-solutions/

---

## ✨ Features

- **Responsive Design** - Mobile, tablet, and desktop optimized
- **Semantic HTML5** - Proper structure and accessibility
- **Pure CSS3** - No frameworks, lightweight and performant
- **Interactive Elements** - Modal forms, FAQ accordion, smooth scrolling
- **Modern UI** - Professional design with dark blue branding
- **Team Section** - Display team members with photos
- **Client Testimonials** - 5-star reviews with avatars
- **Quote Modal** - Contact form for service inquiries
- **Smooth Navigation** - Scroll-to-top button and sticky navbar

## 🚀 Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/muhangiemmabaguma-code/corecleen-solutions.git
   ```

2. **Open locally:**
   ```bash
   cd corecleen-solutions
   open index.html
   ```

3. **Or run a local server:**
   ```bash
   python -m http.server 8000
   ```
