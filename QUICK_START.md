## 🎨 Quick Customization

### Change Product Name
Edit `index.html` line 25:
```html
<!-- Change this -->
<div class="logo">CloudFlow</div>
<!-- To your product name -->
<div class="logo">YourProduct</div>
```

### Change Colors
Edit `styles.css` lines 8-9:
```css
--primary-color: #6366f1;      /* Your primary color */
--secondary-color: #ec4899;    /* Your secondary color */
```

### Change Hero Title & Subtitle
Edit `index.html` lines 36-37:
```html
<h1 class="hero-title">Your New Title Here</h1>
<p class="hero-subtitle">Your subtitle text here</p>
```

### Add Your Logo/Images
1. Add images to `/assets` folder
2. Update HTML to reference them:
```html
<img src="assets/your-image.png" alt="Description">
```

## 🐛 Having Issues?

### Blank page?
- Check if `index.html` file is in the correct folder
- Try opening with `file://` protocol
- Or use a live server (Python/Node)

### Styles not loading?
- Clear browser cache (Ctrl+Shift+Delete)
- Make sure `styles.css` is in the same folder
- Check browser console for errors

### JavaScript not working?
- Ensure `script.js` is in the same folder
- Check browser console (F12 → Console tab)
- Disable browser extensions that might block JS

### Modal not opening?
- Check browser console for errors
- Make sure JavaScript is enabled
- Try a different browser
